еҸҳйҮҸеј•з”Ё

з»„жҲҗдёҖдёӘpackageзҡ„еӨҡдёӘж–Үд»¶пјҢзј–иҜ‘еҗҺе®һйҷ…дёҠе’ҢдёҖдёӘж–Үд»¶зұ»дјјпјҢз»„жҲҗеҢ…зҡ„дёҚеҗҢж–Үд»¶зӣёдә’д№Ӣй—ҙеҸҜд»ҘзӣҙжҺҘеј•з”ЁеҸҳйҮҸе’ҢеҮҪж•°пјҢдёҚи®әжҳҜеҗҰеҜјеҮә

goдёӯеҜ№дәҺеҸҳйҮҸ(еҮҪж•°д№ҹжҳҜеҰӮжӯӨ)дҪҝз”ЁйҰ–еӯ—жҜҚеӨ§е°ҸеҶҷжқҘеҢәеҲ«жҳҜеҗҰеҸҜеј•з”Ё. йҰ–еӯ—жҜҚеӨ§еҶҷеҲҷиғҪиў«е…¶е®ғеҢ…еј•з”Ё(дёҚе®Ңе…ЁжӯЈеёё, зӣ®еүҚе…Ҳиҝҷд№ҲзҗҶи§Ј).

iota

iotaеңЁconstдёӯжҜ”иҫғзү№еҲ«, еӣ жӯӨзү№ж„ҸжҸҗеҮәжқҘиҜҙдёҖдёӢ

йҰ–е…ҲпјҢiotaжҳҜеҸҜд»ҘеҸӮдёҺиҝҗз®—зҡ„пјҢ iotaеёҰиЎЁиҫҫејҸзҡ„йғҪе°Ҷеә”з”ЁеҲ°дёӢйқўзҡ„еёёйҮҸ

жҜ”еҰӮдёҠйқўзҡ„1<< iotaпјҢжӯӨж—¶зӯүдәҺ0, ж•ҙдёӘиЎЁиҫҫејҸзҡ„еҖјдёә1

жҜҸдәҢиЎҢ, iotaзҡ„еҖјзӯүдәҺ1дәҶ, 1 << 1, ж•ҙдёӘиЎЁиҫҫејҸзҡ„еҖј2

д»ҘжӯӨзұ»жҺЁ.

iotaеңЁconstе…ій”®еӯ—еҮәзҺ°ж—¶е°Ҷиў«йҮҚзҪ®дёә0(constеҶ…йғЁзҡ„з¬¬дёҖиЎҢд№ӢеүҚ)пјҢд»Һз¬¬дёҖиЎҢејҖе§Ӣ, жҜҸж–°еўһдёҖиЎҢеёёйҮҸеЈ°жҳҺ(дёҚеҢ…жӢ¬з©әиЎҢеҸҠжіЁйҮҠ)е°ҶдҪҝiotaи®Ўж•°дёҖж¬Ў(iotaеҸҜзҗҶи§ЈдёәconstиҜӯеҸҘеқ—дёӯзҡ„иЎҢзҙўеј•)гҖӮдҪҝз”ЁiotaиғҪз®ҖеҢ–е®ҡд№үпјҢдё»иҰҒз”ЁдәҺе®ҡд№үжһҡдёҫпјҢиҖҢдё”еҫҲжңүз”Ё

constеҸҠiotaеҸҜеҸӮиҖғиҝҷйҮҢ

еёёи§Ғеҹәжң¬зұ»еһӢйӣ¶еҖј

жңүдёӘжҜ”иҫғзү№ж®Ҡзҡ„йӣ¶еҖјnil, иҝҷдёӘеҝ…йЎ»иҰҒеҚ•зӢ¬иҜҙ.

жңҖејҖе§Ӣзҡ„ж—¶еҖҷдёҖи„ёиҢ«з„¶, еҫҲжҳҺжҳҫжҳҜеӣ дёәеңЁideиҝҗиЎҢзҡ„ж—¶еҖҷпјҢжү§иЎҢзҡ„е‘Ҫд»ӨжҳҜgo run main.go, еҸӘдј е…ҘдәҶдёҖдёӘmain.go,иҖҢжІЎжңүдј йҖ’others.go, еӣ жӯӨжүҫдёҚеҲ°mainxx(), еңЁе‘Ҫд»ӨиЎҢдёӢжү§иЎҢд№ҹдјҡжҠҘеҗҢж ·зҡ„й”ҷиҜҜ. йңҖиҰҒиҝҗиЎҢgo run .жҲ–иҖ…go run *.go,еҸҰеӨ–, жңҖеҘҪдёҚиҰҒеҪўжҲҗеҫӘзҺҜеј•з”Ё, еҰӮжһңдёӨдёӘж–Үд»¶йңҖиҰҒзӣёдә’еј•з”Ёзҡ„иҜқ, еј•з”Ёз¬¬дёүж–№жҳҜдёӘжҺЁиҚҗзҡ„еҠһжі•.
