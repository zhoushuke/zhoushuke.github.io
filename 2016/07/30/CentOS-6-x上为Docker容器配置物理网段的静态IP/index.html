CentOS 6.x上为Docker容器配置物理网段的静态IP

发表于 2016-07-30 分类于 CloudNavi 阅读次数： 本文字数： 2.4k 阅读时长 ≈ 2 分钟

虚拟机安装centos6.5(ip范围从192.168.150.130-255，虚拟机网关192.168.150.2)

实验步骤:

0.停止docker:
service docker stop

1.安装brctl
yum install bridge-utils

2.配置桥接接口

brctl addbr br0

brctl addif br0 eth0

3.这是br0的配置文件ifcfg-br0:

vim /etc/sysconfig/network-script/ifcfg-br0（可复制ifcfg-eth0）

4.这是eth0的配置文件ifcfg-eth0:

vim /etc/sysconfig/network-script/ifcfg-eth0

5.重启网络:
service network restart
ip addr

ifcofnig

6.配置Docker使用br0

7.修改docker变量使用br0:

vim /etc/sysconfig/docker
other_args="-b=br0"

8.启动docker:service docker start
查看 iptables是否正常：
iptables -t nat -L -n

9.启动docker容器并设置ip:
启动docker容器：

docker attach

查看ip addr 只有一个lo
